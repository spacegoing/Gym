# RFC (Request for Comments)

This directory contains internal design discussions, proposals, and analysis documents that are **not built** as part of the public documentation.

## Purpose

- Document design decisions before implementation
- Record analysis and recommendations for future work
- Provide context for architectural choices

## Files

| File | Status | Description |
|------|--------|-------------|
| `training-tutorials-restructure.md` | Draft | Analysis of training tutorials section and recommendations |
| `documentation-priority-by-code-maturity.md` | Draft | Prioritization of new docs based on code maturity analysis |

## Note

This directory is excluded from the Sphinx build via `exclude_patterns` in `conf.py`.
